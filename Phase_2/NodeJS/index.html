Node Packages

To access any node package we have to
  1. We have to list the package (if applicable) in the package.json file and then install them (using npm i)

  2. We have to let the application know that we will be using that package, for example by writing const fs = require('fs')

FS - file system availble at run time (part of the node library)

readFile is a built in method for accessing and reading the contents of a file.

fs.readFile('path to the file', 'encoding type (usually utf8)', function)

writeFile is another method coming from the fs package for creating a file if it doesn't exist, or replaces the content of an existing file

fs.writeFile('name of the file', 'content to be written into the file', function)

appendFile is another method coming from the fs package for creating a file if it doesn't exist, or adds additional content to an existing file

fs.appendFile('path of the file', 'content to be added into the file', function)

Modularization

Modularization is the activity of dividing a product or system into interchangeable modules.

We can use custom JavaScript files as dependencies for our code


We can write custom code in a JavaScript file and then call on it to access that code

We would require it (i.e. const custom = require('path to custom script'))

Also, in the custom script file we need to export the code using module.exports = { variables, functions, etc }

Switch, case, break, default

It is another way of writing an if else statement. When the keyword switch is invoked, we would supply it with a paramter that we would check if it is true or false.

The we write case statements to check against the value of the parameter defined earlier. Lastly, when the logic is true, we must use break to exit that logic of the switch/case statement, otherwise we move on to the next statement.

A default statement is needed at the end to account for a case not covered by each statement.

Switch (parameter) { case 'value' : "do this" ; break; default: "do this instead" }


Note: default data type from command line is string, we can use parseInt() to convert these entries into numerical values
